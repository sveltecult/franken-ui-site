## Usage

<span class="uk-label uk-label-primary">Note</span> If you are not using Tailwind
CSS, you can skip this step. On your `tailwind.config.js` file, add the Spinner component
within the `ui()` plugin.

```javascript
import ui from "franken-ui";

ui({
    components: {
        spinner: {
            hooks: {}
        }
    }
}),
```

To create a spinner, add the `uk-spinner` attribute to a block element.

```html
<div uk-spinner></div>
```

```example
<div uk-spinner></div>
```
