# Label

<p class="uk-text-lead uk-paragraph">{`Indicate important notes and highlight parts of your content.`}</p>

## Usage

<span class="uk-label uk-label-primary">Note</span> If you are not using Tailwind
CSS, you can skip this step. On your `tailwind.config.js` file, add the Label component
within the `ui()` plugin.

```javascript
import ui from "franken-ui";

ui({
    components: {
        label: {
            hooks: {}
        }
    }
}),
```

To create a label, just add the `.uk-label` class to a `<span>` element.

```html
<span class="uk-label"></span>
```

<Example component="label" index={1} />

## Style modifiers

Add one of the following classes for additional styles.

| Class                 | Description               |
| --------------------- | ------------------------- |
| `.uk-label-primary`   | Adds a primary style.     |
| `.uk-label-secondary` | Adds a secondary style.   |
| `.uk-label-danger`    | Adds a destructive style. |

```html
<span class="uk-label-success uk-label"></span>
```
