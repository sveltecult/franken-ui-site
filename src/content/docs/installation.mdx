## Installation via NPM

Because Franken UI is a Tailwind CSS plugin, it can be included into any existing Tailwind CSS project. To get started, you first need to make sure that you have a [working Tailwind CSS](https://tailwindcss.com/docs/installation) project installed and that you also have Node and NPM installed on your machine.

<span class="uk-label uk-label-primary">Note</span> If your project needs responsiveness,
you also need to install PostCSS to properly sort media queries.

**1\.** Install the latest version of Franken UI using NPM:

```sh
npm install franken-ui
```

**2\.** Include <code class="uk-codespan">presetQuick()</code> inside the presets property on your <code class="uk-codespan">tailwind.config.js</code> file:

```javascript
import presetQuick from "franken-ui/shadcn-ui/preset-quick";

/** @type {import('tailwindcss').Config} */
export default {
  presets: [presetQuick()],
};
```

Or, if you want full control of what components to use, under the hood, this is what the <code class="uk-codespan">presetQuick()</code> function does. Use this configuration as your base and adjust it according to your needs by commenting out components.

<span class="uk-label uk-label-primary">Note</span> It is recommended to do this
one by one to ensure that other parts of the application are not affected, as some
components rely on others to function properly.

**3\.** Finally, install <code class="uk-codespan">postcss-sort-media-queries</code> and <code class="uk-codespan">postcss-combine-duplicated-selectors</code> and update your <code class="uk-codespan">postcss.config.cjs</code> file.

```sh
npm install postcss-sort-media-queries postcss-combine-duplicated-selectors
```

In your <code class="uk-codespan">postcss.config.cjs</code> file, add the following configuration:

```javascript
module.exports = {
  plugins: [
    require("tailwindcss"),
    require("postcss-sort-media-queries")(),
    require("postcss-combine-duplicated-selectors")({
      removeDuplicatedProperties: true,
    }),
  ],
};
```

With this setup, your build process will remove unused CSS classes, eliminate duplicated rules, and properly sort media queries, resulting in a smaller and more efficient CSS file.

For advanced users, exploring [hooks](/docs/hooks) offers a deeper understanding of how to override styles globally. While you can always rely on Tailwind CSS utility classes, hooks provide an alternative approach that can further reduce CSS file sizes and prevent the cluttering of your HTML with utility classes.

## UIkit JavaScript

Once you have installed Franken UI, include the JavaScript files on your page by adding them to the <code class="uk-codespan">&lt;head&gt;</code> section. You can also choose to use the defer attribute to delay script execution. Or, you can place the code before the end of your <code class="uk-codespan">&lt;body&gt;</code> tag.

```html
<script src="https://cdn.jsdelivr.net/npm/uikit@3.21.5/dist/js/uikit.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/uikit@3.21.5/dist/js/uikit-icons.min.js"></script>
```

To set a theme, configure the <code class="uk-codespan">presetQuick()</code> function to use your desired theme. Here's an example using the green theme:

```javascript
import presetQuick from "franken-ui/shadcn-ui/preset-quick";

/** @type {import('tailwindcss').Config} */
export default {
  presets: [presetQuick({ theme: "green" })],
};
```

Or, if you are using the fine-grained configuration, you can set the theme inside <code class="uk-codespan">hooks()</code> and <code class="uk-codespan">variables()</code> functions:

```javascript
import variables from "franken-ui/shadcn-ui/variables";
import hooks from "franken-ui/shadcn-ui/hooks";

const shadcn = hooks({
  theme: "green",
});

/** @type {import('tailwindcss').Config} */
export default {
  plugins: [
    variables({
      theme: "green",
    }),
  ],
};
```

<span class="uk-badge uk-badge-danger">v0.0.13</span> Franken UI supports theme generators. You can refer to this [documentation](/docs/theming#using-theme-generators) on how to generate your own palette.

## Customizing

If there are specific styles you'd like to override, you can always utilize Tailwind CSS utility classes. For more advanced customization options, consider exploring the use of [hooks](/docs/hooks).

## Editor setup

Keeping track of hooks and configurations can be challenging, but with Franken UI's TypeScript support and a good editor, you can benefit from autocomplete features. If you already have Tailwind CSS IntelliSense installed, you won't need any additional plugins. This not only saves time but also eliminates the need to manually look up and type all UIkit classes.
