# About

Franken UI is a port of [shadcn/ui](https://ui.shadcn.com) for plain HTML or server rendered pages. It is compatible with [UIkit 3's](https://getuikit.com) JavaScript and accessibility features. It works both as a standalone or as a Tailwind CSS plugin.

This project is maintained by [sveltecult](https://x.com/sveltecult) and I am a completely independent developer and have no affiliation with Svelte, @shadcn, UIkit and YOOtheme, or any other entities.
