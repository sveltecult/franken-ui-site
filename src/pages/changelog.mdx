# Changelog

<p class="uk-text-lead uk-paragraph">Latest updates and announcements.</p>

## v0.0.13 (June 8, 2024)

- Add support for [theme generators](/docs/theming#using-theme-generators) or watch the demo on [Mastodon](https://mas.to/@sveltecult/112556172122401221) on how to configure your own palette.
- Add [Stepper](/docs/stepper) component.
- Fix `.uk-list` color modifiers.
- Internal changes such as exposing the entire `shadcn-ui` path instead of listing subpaths, `hexes.js` now exports function to convert HSL to hex instead of hardcoded hex.
- Upgrade UIkit from `3.20.8` to `3.21.5`.

## v0.0.12 (May 20, 2024)

- Add [Typography](/docs/typography) classes
- Fix `uk-divider` height
- Fix `uk-description-list-divider` wrong border color and hook
- Remove dead classes in card component
- Refactor docs to use MDX for better maintenance, syntax highlighting, formatting and copying and pasting
- Refactor docs to have better examples that works for both as a standalone or as a Tailwind CSS plugin
- Fix modal width
- Fix missing hooks and components in `presetQuick()`
- Change default navbar gap to `gap-6`

## v0.0.11 (May 11, 2024)

- Introducing the [CDN installation](https://github.com/sveltecult/franken-ui-releases).
- Fix missing hooks and components in `presetQuick()`
- Add sensible padding for modal, table, tab components to make way for CDN aka "no-build" installations. If you are having trouble overriding some paddings, you can always use the not important utilities `!p-6` or consider using [hooks](/docs/hooks).
- Change buttons' shadow from `shadow` to `shadow-sm`
- Add rounded corners on modal even on xs devices
- Fix notification alignment on xs devices
- Change UIkit breakpoints to match Tailwind CSS. Here are the list of changes:

| **Screen** | **Original** | **New** |
|------------|--------------|---------|
| SM         | 640px        | 640px   |
| MD         | 960px        | 768px   |
| LG         | 1200px       | 1024px  |
| XL         | 1600px       | 1280px  |

## v0.0.10 (May 1, 2024)

- This release focus mainly on ease of use, code cleanup and introduction of [`presetQuick()`](/docs/installation)
- Fix `uk-accordion-title` alignment
- Fix `uk-tab-left` and `uk-tab-right` wrong border color
- Refactor `hooks.ts` for better maintenance
- Demystifying [hooks](/docs/hooks)

## v0.0.9 (April 30, 2024)

- Add tag and PIN inputs
- Increase default padding for `.uk-nav-primary` and `.uk-nav-secondary`
- Fix height and line-height of `.uk-button-text` and `.uk-button-link`
